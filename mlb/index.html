+EV Parlays

Cross-game combos built only from legs that individually beat the no-vig consensus — the joint edge compounds the line-shop edge, it doesn't manufacture one. Ordinary parlays multiply vig and are always -EV; this tab is empty unless 2+ real edges exist today. ONE BOOK combos are placeable as a single ticket; "split" combos are better bet as singles.

Home Run Board — best HR bets of the day

P(player homers today) = shrunk season HR/PA × starter's HR-allowed factor (62% weight) × HR-specific park index × live temperature. Ranked by EV when props price in, else by model HR%. Props carry 8-15% hold — a gap vs fair is the entry ticket, not the win. Lineups are season-usage projections until cards post (~1-4h pre-game); a scratch voids that row.

Team Ratings

O = offense multiplier (runs scored vs league), D = defense (runs allowed; below 1 is good), net = O/D. Sample-shrunk season rates — deliberately simple; opponent-adjusted iteration overfit and was rejected.

Backtest (walk-forward, leak-free)

Every prediction made only from games before it. This is a read, not a market-beater: closing moneylines hit ~57-60%; this model is validated below that. Its use is understanding matchups and pricing totals — the bettable edge lives in the Edges tab.

What's in the number

Win%: team run rates (shrunk) + starting-pitcher quality via season xwOBA (725 pitchers), Poisson game grid, home field ~1.03. Rolling per-game xwOBA was tested against the crude runs-allowed proxy on 388k pitches and added nothing — both 55.3% — so the simpler input stands.

Totals: model total × park factor (Statcast 3-yr run index; renovated parks shrunk toward neutral and tagged low-conf rather than dropped) × weather (Open-Meteo live: ~+1%/10°F above 70, wind >8mph; retractable roofs halved, domes zeroed — every game tagged with what applied).

Edges: per-book devig → median no-vig consensus → best price per side → quarter-Kelly on the price gap. In-progress games skipped. Same architecture validated in the UFC tool.

HR board: shrunk batter HR/PA (K=130 PA prior) × starter HR-per-batter factor (K=200 BF, 62% weight, bullpen = league) × HR-specific park index (seed approximations, conf-shrunk like the run parks) × temperature (+0.8%/°F over 70, roof-halved, dome-off; vs a flat 70°F baseline, so warm-climate open parks mildly double-count). Wind and platoon splits deliberately excluded from v1 — wind needs direction × park geometry, platoons need a second pull. Selftested: shrinkage, park/temp/pitcher monotonicity, EV wiring.
